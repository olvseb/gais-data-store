# gais-data-store

Data store for the [GAIS Data](https://olvseb.github.io/gais-data/) fan website.

## Data source

Match data, league tables, and statistics are sourced from [svenskfotboll.se](https://www.svenskfotboll.se/) (Svenska Fotbollförbundet) and [Fogis](https://fogis.se/).

This is a non-commercial fan project focused on GAIS. Data is displayed in unmodified factual form. Team logos are property of their respective clubs.

This site has no affiliation with SvFF or GAIS.
